PROJECT: How are you feeling?

Ready for some facial expression recognition? During my first semester at my bachelors we made this fun little project. The project examined how well a convolutional neural network could detect different facial expressions compared to humans. Turns out it could perform on par with us!

PROJECT: What do you mean?

Let's apply some Natural Language Processing to better understand eachothers points. Drop a text below (it does not get stored) and check out the structure of the text as well as what aspects and sentiments are the most important in the sentences.

PROJECT: Hey, can you help me match this piece of jewellery?

Sure I can! In this project we used Content Based Image Retrieval to create a recommendation system that can be used in e-commerce related settings. It could be implemented both as a search-by-image model as well as a recommendation engine. I sometimes still get spooked whenever an ad for jewellery pops up on my social media - it gives my flashbacks to many long nights in front of Visual Studio Code.
